```
สวัสดีครับ อยากสอบถามหลายเรื่องเลยนะครับ ขอถามรวดเดียวเลยแล้วกัน

อย่างแรกเลยคือ LazyHardWork รับทำอะไรบ้างครับ มีบริการหลักกี่อย่าง แต่ละอย่างทำอะไรได้บ้าง อย่างที่สองคืออยากรู้เรื่องราคาครับ ถ้าจะทำ LINE OA chatbot แบบง่ายๆ ก็พอ ราคาเริ่มต้นเท่าไหร่ แล้วแพ็ก Starter กับ Advanced ต่างกันยังไง อย่างที่สามคืออยากรู้ว่าทำเสร็จกี่วันครับ ถ้าสั่งงานวันนี้จะเสร็จภายในกี่วัน และอย่างสุดท้ายคืออยากดูผลงานของ LazyHardWork ก่อนตัดสินใจครับ มีผลงานที่ไหนให้ดูบ้าง และมีรีวิวจากลูกค้าเก่าไหมครับ ขอบคุณมากครับ

```

| E-05 | ส่ง `สวัสดี` → ทันทีส่ง `ถามเรื่องราคา` → ทันทีส่ง `แล้วก็ผลงาน` (ภายใน 3 วิ) | บอทตอบครั้งเดียวหลัง ~6.5 วิ · ไม่ตอบ 3 รอบแยก |
