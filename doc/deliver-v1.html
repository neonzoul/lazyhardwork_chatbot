State Machine (วิธีทำงานของบอท)

  ลูกค้าส่งข้อความ
       │
       ▼
  ┌────────────┐    handoff trigger     ┌───────────────┐
  │ BOT_ACTIVE │ ──────────────────────► WAITING_HUMAN  │
  │  บอทตอบปกติ│                        │  บอทหยุดตอบ  │
  └────────────┘                        │  #crm แจ้ง Mos│
       ▲                                └───────┬───────┘
       │                                        │
       │  admin set-status → BOT_ACTIVE         │ Mos กด HUMAN_ACTIVE
       │                                        ▼
       │                                ┌───────────────┐
       └────────────────────────────────  HUMAN_ACTIVE  │
            admin set-status            │  บอทเงียบ     │
                                        │  Mos ตอบเอง  │
                                        └───────────────┘

handoff trigger มี 3 ช่องทาง

  • กดปุ่ม Rich Menu "คุยกับคนจริง"
  • พิมพ์ว่า "ขอคุยกับคน" / "คุยกับมอส" / "อยากนัดคุย"
  • บอทตอบไม่ได้ (KB gap) → ถามลูกค้า → ลูกค้ายืนยัน → handoff อัตโนมัติ

วิธีใช้งานประจำวัน

เมื่อได้รับ Discord alert #crm

  1. เปิด LINE OA → ไปที่ chat ของลูกค้าคนนั้น
  2. ตอบลูกค้าโดยตรง (บอทเงียบอยู่แล้วในสถานะ WAITING_HUMAN)
  3. เมื่อพูดคุยเสร็จ → เปิด Admin Dashboard → เปลี่ยนสถานะกลับเป็น BOT_ACTIVE
Admin Dashboard อยู่ที่ /admin?secret=<ADMIN_SECRET> — เพิ่ม bookmark ไว้เลย

เมื่อได้รับ Discord alert #leads

  1. ดูข้อมูล lead ที่บอทส่งมา (ชื่อ / ธุรกิจ / ปัญหา / งบ / ช่องทางติดต่อ)
  2. ติดต่อลูกค้ากลับตามช่องทางที่ระบุ
  3. บอทยังคุยกับลูกค้าต่อไปได้ระหว่างที่รอ Mos

วิธีอัปเดต Knowledge Base

อัปเดต KB = restart server ทุกครั้ง เพราะ KB โหลดตอนเปิดเครื่อง
  1. เปิดไฟล์ knowledge/kb.md
  2. แก้ไขข้อมูลที่ต้องการ (ราคา / บริการ / FAQ / เวลาทำการ)
  3. บันทึกไฟล์
  4. Restart server: npm run dev หรือ redeploy บน Render
ไม่ต้องแตะ code แม้แต่บรรทัดเดียว — KB คือไฟล์ text ธรรมดา

Admin Endpoints

การ Deploy

Render (แนะนำ)

  1. เชื่อม GitHub repo → Render auto-deploy เมื่อ push
  2. ใส่ env vars ทุกตัวใน Render dashboard
  3. อัปเดต LINE webhook URL → HTTPS URL จาก Render + /webhook
Free tier หยุดเมื่อไม่มี traffic 15 นาที — upgrade เป็น Starter ($7/เดือน) เพื่อให้ทำงานตลอด 24 ชั่วโมง
